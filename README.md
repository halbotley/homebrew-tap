# Hal Botley's Homebrew Tap

Swift CLI tools for macOS by [Hal Botley](https://github.com/halbotley).

## Installation

```bash
brew tap halbotley/tap
brew install calmly timely
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| **calmly** | A calm CLI for macOS Calendar. No dialogs, no prompts, just events. |
| **timely** | A CLI for Apple Reminders with geofencing support. |

## Requirements

- macOS 13.0+
- Xcode Command Line Tools
